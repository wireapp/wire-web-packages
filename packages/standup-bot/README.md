## Standup Bot

A bot that can add people to a certain chat.

### Getting Started

```
yarn
yarn start -e bot@wire.com -p password -c e792358c-faa1-4b60-b5e0-96bcafe9e13b
```

### Usage

Just ask for `--help` to get informed on how to use this bot:

**Node**

```bash
yarn dist
node dist/start.js --help
```
