# prettier-config

> Wire's [Prettier](https://prettier.io) config.

## Usage

**Install**:

```bash
$ yarn add --dev @wireapp/prettier-config
```

**Edit `package.json`**:

```jsonc
{
  // ...
  "prettier": "@wireapp/prettier-config"
}
```
